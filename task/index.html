Api Name Api Description
Country Code

The iso country code of any given point. User must insert a latitude (a number between -90 and 90) and a longitude (a number between -180 and 180)
Ocean

The name of the ocean or sea for the given latitude/longitude.
Timezone

The timezone at the lat/lng with gmt offset (1. January) and dst offset (1. July).

(Radius adds buffer in km for closest timezone in coastal areas)
