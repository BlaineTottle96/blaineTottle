About

A simple one-page, full screen responsive application with Bootstrap and Leaflet.

Features
  • Fullscreen mobile-friendly map with responsive navbar and modal placeholders
  • jQuery ajax call to PHP curl routines returning third party API Information
  • jQuery onload retrives users current location using leaflets inbuilt setView function
  • Search Bar containing country options returning border information
  • Responsive sidebar with links to modals populated with API data for selected country
  • Wikipedia marker icons included in grouped layer control updated on map for selected
